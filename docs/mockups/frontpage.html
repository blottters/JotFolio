Summary

A compiler reads Raw Notes from the inbox and emits durable Wiki Entries that retrieval prefers over raw material. See Karpathy Talk Capture for the source observation.

Mechanism

Each compiled entry tracks provenance (source ids), confidence (heuristic), and supersedes (prior compiled entries this replaces). Ghost notes like Stale Knowledge Decay show as dashed circles until you create them.

Raw stays the truth. Compiled is derived. Manifest is the only thing that knows the difference.

Open questions

  • Does Affinity Layout work better than clusters for retrieval surfaces?
  • Should review entries auto-graduate to wiki when confidence ≥ 0.8?
