# JotFolio

JotFolio is a local-first mixed-media knowledge workspace for notes, videos,
podcasts, articles, journals, links, canvases, saved views, and relationship
mapping.

The product direction is privacy-first and vault-centered: user content is meant
to stay portable, inspectable, and organized around durable markdown files rather
than a closed database.

## Current Status

JotFolio is in alpha. The current source line is `0.5.0-alpha.15`.

This repository is under active development. Public documentation is intentionally
minimal while the product model, release process, and desktop distribution are
still being stabilized.

## Core Ideas

- Local-first vault ownership
- Mixed-media entry capture
- User-authored keyword automation
- Saved views and canvases for workflow organization
- Relationship mapping for synthesis
- Opt-in diagnostics only

## Documentation

Detailed development, release, security, and historical planning notes are kept
for maintainers and are not part of the public-facing overview.
